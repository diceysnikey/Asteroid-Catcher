## Info:
An endless arcade-like game made in Godot 4 for Hackclub's Flavortown. 
To raise your score, catch green asteroids while dodging the red ones! Use the spacebar for a boost

## Features:
- Movement with keyboard
- Highscore saving system
- Fullscreen support
- Music and SFX
- Color blindness mode

## How to run:
Download the .exe for Windows, or the x86_64 executable for Linux. Simply run it to start the game.
